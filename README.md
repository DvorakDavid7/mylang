# mylang

## usage

- run instructions from _.dd_ file
```bash
python3 mylang.py ./examples/avg.dd
```

- run instructions from command line argument
```bash
python3 mylang.py '1 2 3 ADD PRINT'
```

- run **REPL** mode
```bash
python3 mylang.py -r
```


## **REPL** mode
in REPL mode there are some special commands:

- memory dump
```
>>> mem
```

- clear memory
```
>>> cls
```

- exit repl
```
>>> exit
```
